Project Goal

My goal with this project was to design and develop a modular, responsive, and immersive third-person melee combat system featuring proper collision handling, a flexible damage system, dynamic enemy AI, and visual/audio feedback for the player. Inspired by games like Sekiro and Ghost of Tsushima, I wanted to create a smooth system that felt satisfying to play and scalable enough to support a full RPG.

Project Overview

This project showcases a modular third-person melee combat system developed in Unreal Engine 5, designed to demonstrate core gameplay mechanics, AI design, and responsive UI.

The system supports light and heavy attack combos, parry and block mechanics, a shared damage interface, and a fully integrated health and stamina system tied to the UI. Enemies use unique behavior trees, perception systems, and EQS-based decision-making for intelligent combat behavior and movement.

The project was built for scalability and modularity, using Unreal's component-based architecture, interfaces, and clean separation between player and AI logic.

Combat Mechanics

Light & Heavy Attack Combos

Smooth chained animations using montage-based combo logic. Players can switch between light and heavy attacks mid-chain.

Directional Attacks

Attacks respond to player movement direction and aim for more dynamic, readable combat.

Hit Detection

Precise weapon collision using custom trace logic for accurate, responsive hit registration.

Blocking System

Negates incoming damage depending on stamina level and enemy attack type.

Parry System

Parry window controlled via anim notifies. Timed parries trigger hit-stun animations and open enemies to counterattacks.

Damage System

Built around a custom DamageComponent and DamageInterface — attachable to any actor for consistent damage handling with per-actor customisation. Both Player and Enemy implement IDamageInterface for shared function calls while defining their own response logic.

By separating logic through a component and interface-based design, the system supports clean, scalable, and highly modular development — easy to add or update damageable actors without rewriting logic.

Interface Functions

  • TakeDamage() — Accepts incoming damage, triggers hit reactions and VFX
  • Heal() — Restores health up to the actor's maximum
  • SetMaxHealth / SetCurrentHealth — Blueprint or runtime stat setup
  • Die / OnDeath — Actor-specific death behavior
  • GetCurrentHealth, GetMaxHealth, IsDead — Status queries for UI, combat logic, and AI

Enemy AI

The system features multiple enemy classes, each with unique combat behavior and attack patterns. All types are driven by a shared underlying AI architecture but individually configured to behave differently.

  • Fast Attackers — Strike quickly and unpredictably, forcing faster player reactions
  • Strafing Enemies — Actively reposition using EQS before engaging
  • Boss Enemies — Heavy-hitting with larger hitboxes and aggressive AI logic

AI States

Idle · Attack · Frozen (Dazed/Parried) · Strafe

AI Controller

Possesses the enemy pawn on spawn, manages Behavior Tree execution, and converts Perception System updates into Blackboard changes for state transitions.

Behavior Trees

Each enemy class has a unique Behavior Tree driving state switching in response to Blackboard key changes triggered by perception updates.

AI Perception

Sight-based player detection. Sends perception updates to the AI Controller for Blackboard state transitions.

EQS

Generates and scores environment points based on distance, visibility, and player location for dynamic enemy strafing and positioning.

Visual & Audio Feedback

VFX (Niagara)

  • Weapon trails on fast swings to convey motion and power
  • Hit impact effects — sparks and blood bursts on hit events
  • Sharp flashes and burst effects on parry and block
  • Brief flash/particle emission on player and enemy damage

SFX

  • Different swoosh sounds for light and heavy attacks
  • Varying impact sounds depending on hit area
  • Sharp metallic audio cues on successful parry and block
  • Subtle ambient SFX to keep combat grounded in the world
