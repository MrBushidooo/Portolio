Kinetic Aim

Fast-paced 3D FPS target-shooting game with dynamic difficulty scaling and Chaos Physics destruction, backed by a full Game Design Document.

Genre
FPS / Arcade Shooting
Platform
PC (Windows)
Engine
Unreal Engine 5
Tools
Blueprints, Chaos Physics, Blender

About the Game

Kinetic Aim is a 3D first-person target shooting game where players must shoot randomly spawning targets within a time limit to progress through multiple levels. As the game advances, targets move faster and follow more complex patterns. Chaos Physics provides realistic destruction effects on hit, and all 3D assets were created in Blender.

Systems Built

Target Spawning System

Dynamic system spawning targets at random locations, ensuring each session feels fresh and unpredictable.

Scoring & Progression

Players must shoot targets within a time limit to advance levels, with tracked score and clear UI feedback.

Difficulty Scaling

Multiple levels with increasing difficulty — faster and more complex target movement patterns as the player progresses.

Chaos Physics

Realistic destruction effects when targets are hit, powered by UE5's Chaos Physics system.

Blueprint Scripting

Core gameplay — target spawning, hit detection, scoring, level transitions — all built in Blueprints.

3D Asset Creation

Environment and target assets modelled and optimised in Blender, balancing performance with visual quality.

Game Design Document

A full GDD was produced covering the core game loop, level design rationale, difficulty curve planning, and target behaviour specifications.
